# Игра «Элиас» — Telegram-бот

## Описание
Telegram-бот для игры в «Элиас». Игрок объясняет слова своей команде, нажимает кнопки:
- ✅ **Отгадал** — +1 очко
- ⏭ **Пропустить** — слово меняется
- ⏹ **Завершить раунд** — передаёт ход следующей команде

Побеждает команда, первой набравшая **20 очков**.

## Требования
- Python 3.12
- Файл `words.txt` в кодировке UTF-8 (одно слово на строку)
